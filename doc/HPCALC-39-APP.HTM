库删除器  (详情) 39/40 英文   36KB/1KB
这是一个可以删除其他库的库！库是一些程序所带的用机器码编写的附属文件。您可以通过在任意一机型上按SHIFT+MEMORY并选择LAB子目录来查看它们。您大可以删除它们中的大多数，我敢保证，如果没有这个库，就算老天爷来删除了母程序也不能将之删除。这个库将要做的就是这个。您只需在MEMORY视图下记住您所需删除的库之编号姑且记作NUM，之后在主界面输入
UNLIB NUM

如果那个库当时未有被运行，那它既会消失。您不能删除这个库本身，但189B的内存占用简直不是个事儿。 You can't delete the UnLib library itself but since it's only 189 bytes in size that hardly matters.

By Noda. 2003/05/20
